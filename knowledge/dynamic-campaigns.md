# Dynamic Campaigns based on Lead Stage / Custom Property

> 📖 Full article with screenshots: https://crmchat.ai/help-center/dynamic-campaigns

### Introduction

The Dynamic Sequence feature allows you to automate sending a series of personalized messages to your CRM Telegram leads. This is the perfect tool for running automated follow-up campaigns for leads that were assigned a certain custom property or reached a certain CRM stage.

### Step 1: Set up a custom property / CRM stage

Before creating a sequence, you need to prepare the custom properties that you will use to filter your leads. To do this, go to **'Custom Properties'** and set up the relevant properties.

*We've already prepared some default properties, such as **'Stage'**. For instance, if you select **'Stage'** as a filter for your dynamic campaign, all leads moved to that stage in the sequence will be added to the dynamic outreach sequence.*

### Step 2: Creating a Dynamic Sequence

With your custom properties ready, you can now create a dynamic sequence in the platform. You can find this feature in the main navigation under **Outreach** > **Sequences > New Sequence > CRM Leads**. 

Select the custom properties that you want to use for filtering. You can combine two or more filters; for example,** 'Stage' **and **'Industry'**. This will ensure that only leads on that stage and with that industry tag are added to the dynamic sequence.
